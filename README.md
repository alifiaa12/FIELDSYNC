# FIELDSYNC
SIPROMA merupakan website monitoring proyek yang membantu pengawas dan karyawan dalam mengelola progress pekerjaan secara real-time. Sistem ini dilengkapi fitur absensi, upload evidence, monitoring progress harian, generate laporan, serta ekspor data Excel sehingga proses pelaporan menjadi lebih cepat, akurat, dan efisien.
